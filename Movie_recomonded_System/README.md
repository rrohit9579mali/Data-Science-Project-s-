# 🎬 Movie Recommendation System

A content-based movie recommender system built using Machine Learning and Streamlit.  
It suggests similar movies based on user-selected titles.

---

## 🚀 Features
- Select a movie from dropdown
- Get top 9 similar movies
- Movie posters using TMDb API
- Interactive UI with Streamlit

---

## ⚙️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit
- TMDb API

---

## 🔄 Workflow
1. Collected movie dataset  
2. Data cleaning and preprocessing  
3. Created tags (genres, cast, keywords)  
4. Converted text data using CountVectorizer  
5. Calculated similarity using Cosine Similarity  
6. Built recommendation function  
7. Created UI using Streamlit  

---
